# 求职小助手 · 测试使用说明

> 感谢参与测试！本工具包含三个核心模块：简历优化、我的档案、投递追踪，以及一个 Chrome 插件用于网申表单自动填写。

---

## 一、安装 Chrome 插件

1. 解压收到的 `extension.zip`
2. 打开 Chrome，地址栏输入 `chrome://extensions`
3. 右上角开启 **开发者模式**
4. 点击「**加载已解压的扩展程序**」，选择解压后的 `extension` 文件夹
5. 插件栏出现图标即安装成功

### 配置服务器地址

点击插件图标，将底部「服务器地址」改为：

```
http://110.40.140.2:8005
```

## 二、网页端功能

浏览器访问：**http://110.40.140.2:8005**

### 1. 简历优化（首页）

| 步骤 | 操作 |
|------|------|
| ① | 上传 PDF 格式简历 |
| ② | 点击「分析简历」，等待 AI 给出优化建议（约 10–20 秒） |
| ③ | 粘贴目标岗位 JD，点击「生成优化简历」 |
| ④ | 生成完成后点「下载 Word 文件」获取优化后简历 |

---

### 2. 我的档案

用于存储简历之外的补充信息，供插件自动填表使用。

| 步骤 | 操作 |
|------|------|
| ① | 点击「+ 新建」创建档案，上传简历 PDF 自动解析 |
| ② | 检查并补充各分区信息（教育、工作、技能等） |
| ③ | 补充「补充信息」分区：期望薪资、期望行业、自我评价等 |
| ④ | 信息越完整，插件填表越准确 |

支持多档案（如：校招版、社招版），可随时切换。

---

### 3. 投递追踪

| 功能 | 说明 |
|------|------|
| 添加投递记录 | 点击「+ 添加投递」手动记录，或插件填表后自动记录 |
| 看板视图 | 按状态分栏展示（待投递 → 已投递 → 面试中 → Offer / 拒绝） |
| 表格视图 | 支持按公司、状态、平台筛选 |
| 详情面板 | 点击任意记录，可添加面试轮次、备注、状态变更时间线 |
| 数据图表 | 状态分布、每周投递趋势、每日热力图 |

---

### 4. 面试准备

| 步骤 | 操作 |
|------|------|
| ① | 上传简历 PDF |
| ② | 粘贴目标岗位 JD |
| ③ | 点击「生成面试题」，AI 输出高频问题 + 参考答案 |

## 三、插件自动填表

目前支持的网站：

| 网站 | 支持程度 |
|------|----------|
| 智联招聘校园版（xiaoyuan.zhaopin.com） | ✅ 完整支持（8个分区） |
| 猎聘（liepin.com） | ✅ 通用表单 |
| 58同城简历（jianli.58.com） | ✅ 支持 |
| 其他 Element UI 表单网站 | ⚡ 通用适配 |

**使用方法：**
1. 打开目标网站的网申表单页面
2. 点击插件图标，选择对应档案
3. 点击「**一键填入**」
4. 等待自动填写完成（约 10–30 秒）

> ⚠️ 填写完成后请人工检查一遍，确认无误后再提交
